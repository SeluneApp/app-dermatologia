# Selune ִֶָ☾.

## 🚧 EM ANDAMENTO 🚧


O **Selune** é um aplicativo desenvolvido como Trabalho de Conclusão de Curso (TCC), com o objetivo de auxiliar pessoas que possuem problemas de pele.  
O app permite realizar **registros diários** sobre a saúde da pele, acompanhar evolução ao longo do tempo e receber recomendações de produtos indicados.  

---

## ˙⋆✮ Funcionalidades  

- 📅 Registro diário da pele com formulário interativo  
- 🗂 Histórico organizado por calendário  
- 🛍️ Página de produtos recomendados com rolagem vertical e horizontal  
- 🔐 Sistema de login e cadastro de usuários  
- 🌙 Interface pensada para **acompanhamento noturno** (modo escuro e claro)  

---

## ٠࣪⭑ Tecnologias utilizadas  

- [React Native](https://reactnative.dev/) + [Expo](https://expo.dev/)  
- [React Navigation](https://reactnavigation.org/) (`createStackNavigator`)  
- [JavaScript ES6+]  
- Banco de dados: PostgreeSQL (back-end integrado pela equipe)
